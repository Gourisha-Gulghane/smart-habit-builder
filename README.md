# Smart Habit Builder

Smart Habit Builder is a habit tracking system that helps users
create and maintain daily habits.

## Features
- Habit creation
- Daily tracking
- Streak monitoring
- AI habit analysis

## Technologies Used
Python
Flask
Machine Learning
